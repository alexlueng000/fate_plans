# Logo 设计提示词

## 项目信息

为**易占大师/命理洞察平台**（FateInsight）设计品牌logo

- **品牌名称**：易占大师 / FateInsight
- **核心服务**：AI驱动的八字命理分析、风水咨询、六爻占卜
- **目标用户**：25-45岁关注传统文化的现代都市人群
- **品牌调性**：传统智慧与现代科技的融合

---

## 设计要求

### 风格定位
- **现代简约 + 中国传统元素融合**
- 专业感与神秘感并重
- 适合互联网产品的视觉语言

### 配色方案
- **主色**：朱砂红（中国传统吉祥色）
- **辅助色**：墨黑/深灰（稳重、专业）
- **点缀色**：金色（高级感、神秘感）

**色值参考**：
```
朱砂红: #C8102E / rgb(200, 16, 46)
墨黑:   #1A1A1A / rgb(26, 26, 26)
金色:   #D4AF37 / rgb(212, 175, 55)
```

### 必须包含的元素（选其一）
1. **八卦符号**的现代抽象化表达
2. **阴阳太极**的几何简化设计
3. **天干地支**的图形化创意
4. **罗盘/星象**的极简几何形态

### 技术规格
- ✅ 矢量图形，适合任意缩放
- ✅ 在 16x16px favicon 尺寸下仍可识别
- ✅ 黑白版本同样有效
- ✅ 适配深色/浅色背景
- ✅ 适合横版、竖版、方形多种布局

### 避免的元素
- ❌ 过于复杂的细节装饰
- ❌ 陈旧的民俗风格
- ❌ 迷信色彩过重的符号
- ❌ 过度使用渐变和阴影

## 设计迭代流程

### 第一轮：风格探索（生成 3-5 个方向）
1. **方向A**：八卦符号抽象化（几何线条）
2. **方向B**：阴阳太极现代演绎（负空间设计）
3. **方向C**：罗盘元素简化（圆形构图）
4. **方向D**：天干地支符号化（文字图形）
5. **方向E**：星象/命盘几何化（点线面构成）

### 第二轮：细节优化
- 选择最佳方向
- 调整配色比例（朱砂红的使用面积）
- 优化图形比例和平衡感
- 生成横版/竖版/方形变体

###第三轮：应用场景测试
生成以下尺寸的预览：
- **Favicon**: 16x16px, 32x32px
- **App Icon**: 512x512px, 1024x1024px
- **网页 Header**: 200x60px（横版）
- **社交媒体**: 400x400px（方形）
- **深色背景版本**
- **浅色背景版本**

- 保持文化认同但不陈旧

## 输出文件要求

### 文件格式
- **矢量源文件**: AI / SVG / EPS
- **位图导出**: PNG（透明背景）/ JPG
- **Web优化**: SVG（代码优化）

### 尺寸规格
```
favicon.ico       - 16x16, 32x32, 48x48
logo.svg          - 矢量原文件
logo-horizontal.png - 2000x600px
logo-square.png   - 1024x1024px
logo-icon.png     - 512x512px
```

### 颜色版本
- 全彩版（朱砂红 + 金色 + 黑色）
- 单色版（纯黑）
- 反白版（纯白，用于深色背景）

## 品牌应用示例

### 网站应用
- **导航栏**: 横版logo（高度40-50px）
- **登录页**: 方形logo（200x200px）
- **Favicon**: 简化图标版本

### 移动端应用
- **App启动页**: 方形logo居中
- **底部导航**: 简化icon版本
- **微信小程序**: 圆形裁切适配

### 线下物料
- 名片、宣传册
- 易拉宝、海报
- 周边产品（文创）

## 设计师沟通要点

如果委托专业设计师，需要明确：

1. **品牌定位**：不是传统算命摊，而是现代AI命理咨询平台
2. **目标用户**：年轻都市白领，有文化认同但追求现代审美
3. **使用场景**：主要是数字产品（网站、App、小程序）
4. **文化元素**：要有中国传统符号，但必须现代化处理
5. **配色限制**：必须使用朱砂红作为主色调
