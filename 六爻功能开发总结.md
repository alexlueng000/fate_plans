已实现5个核心API端点：

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/liuyao/paipan` | POST | 六爻排盘 |
| `/api/liuyao/history` | GET | 获取历史记录列表 |
| `/api/liuyao/{hexagram_id}` | GET | 获取单个卦象详情 |
| `/api/liuyao/{hexagram_id}` | DELETE | 删除卦象记录 |

## 核心算法说明

### 数字起卦
```
上卦 = (数字1 % 8) 或 8
下卦 = (数字2 % 8) 或 8
动爻 = (数字3 % 6) 或 6
```

### 铜钱起卦
```
摇6次，每次3个铜钱
正面=3，反面=2
总和6=老阴（变），7=少阳，8=少阴，9=老阳（变）
```

### 时间起卦
```
上卦 = (年+月+日) % 8
下卦 = (年+月+日+时) % 8
动爻 = (年+月+日+时) % 6
```
