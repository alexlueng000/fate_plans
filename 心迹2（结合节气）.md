# 《心镜灯》产品设计文档（完整版）

**副标题**：顺着天地的自我对话  
**核心定位**：一个与自己对话的私密空间，用天地接纳情绪，用叙事觉察成长

**版本**：6.0（完整版）  
**最后更新**：2026年4月

---

## 一、产品核心理念

| 维度         | 描述                                       |
| ------------ | ------------------------------------------ |
| **对用户**   | 不是算命，不是治疗，是一个与自己对话的空间 |
| **情绪归因** | “今日节气如此”而非“你有问题”               |
| **技术底色** | ACT接纳承诺疗法 + 叙事治疗，不越界         |
| **数据主权** | 所有记录只在用户设备，不上传               |
| **语言风格** | 轻量、中性、不评判、不触发防御             |

**每次打开心镜灯首页的提示**：
> 天地有节气，人心有潮汐。  
> 所有记录只留在你这里。

### 模块二：心迹（轻量接触）

#### 2.1 身体感受

**提示**：“身体是情绪的锚。看见就好。”

**图标**：
- 胸口
- 肩膀
- 胃
- 其他地方
- 都还好

**交互**：点击即记录。

| ---- | ---- | ---------------------------------------- |

| 😤    | 伤官 | “它来了。它来的时候，想说什么？”         |
| 😞    | 正官 | “它来了。它来的时候，在想什么？”         |
| 🏠    | 偏印 | “它来了。它来的时候，想去哪里？”         |
| 🍂    | 劫财 | “它来了。它来的时候，想做什么？”         |
| 🌪️    | 枭神 | “它来了。它来的时候，思绪去了哪里？”     |

**非命理版**：

| 图标 | 名称 | 引导语                                   |
| ---- | ---- | ---------------------------------------- |

| 😞    | 内疚 | “它来了。它来的时候，在想什么？”         |

#### 2.3 它想说什么

**提示**：“有些话说不出口。可以写在这里。”

**交互**：
1. 文本框：“写一句你说过或想过的矛盾话”
2. 点击“看看它想说什么”
3. 系统显示：

> “那不是你，是心里一个声音。它可能想：”
> - “怕靠太近”
> - “怕欠人情”
> - “怕自己不够”
>
> “你可以听听它，也可以做自己的。”

**预设库**（随机3条）：

| 输入       | 可能翻译                         |
| ---------- | -------------------------------- |
| “别对我好” | 怕靠太近 / 怕欠人情 / 怕自己不够 |
| “我没事”   | 怕麻烦人 / 怕被看穿 / 怕说太多   |
| “我不在乎” | 怕在乎 / 怕失控 / 怕被拒绝       |
| “你好烦”   | 怕太近 / 怕被绑 / 怕心动         |
| “你走吧”   | 怕你走 / 怕你留 / 怕留不住       |
| “不用管我” | 怕成为负担 / 怕被怜悯 / 怕依赖   |

---

#### 2.4 小记

**提示**：“过去7天，有没有一次，不太一样？”

**交互**：
- 每周出现一次
- 两个按钮：【有】【没有】
- 点击【有】后：
  - 文本框：“那次发生了什么？可以写几个字。”
  - 保存后显示：“这个时刻在[节气]。天地有缝隙，你看见了。”

**存储**：保存所有小记。

---

#### 2.5 一周图

**图表内容**：
- 横轴：最近7天
- 灰色柱：“来了什么”的次数
- 橙色点：身体感受的记录
- ✨：小记的日子
- 💡：深度对话的日子

**图上文字**：“这是这一周的潮汐。潮水有涨落，你是看海的人。”

**下方注释**：“本周[节气]。天地在转，你也在。”

**交互**：点击✨看小记，点击💡看对话概要，点击日期看当天记录。

---

#### 2.6 可以试试

**提示**：“选一个场景。看看可以试什么。”

**场景**（可自定义）：
- “有人想约我见面”
- “我想联系一个人”
- “我需要做一个决定”
- “其他：______”

**两个问题**：
1. “如果不被它带着走，你会想怎么做？”
2. “从那里，找一件很小的事，这周可以试试吗？”

**推荐**（基于节气）：

| 节气 | 推荐语                                 |
| ---- | -------------------------------------- |
| 春   | “春天。可以试试一件新的事，很小就行。” |
| 夏   | “夏天。可以试试把心里的话写下来。”     |
| 秋   | “秋天。可以试试少做一件事。”           |
| 冬   | “冬天。可以试试什么都不做，只是待着。” |

**保存**：“这周，我可以：______”

**下周**：可标记【试了】【没试】。系统回应：“试了就好。”“没试也正常。可以再调小一点。”

### 模块三：夜灯（深度对话）

**入口提示**：
> 这里是你的空间。系统只问问题，不分析。  
> 可以随时退出。

---

#### 3.1 走一段路

**形式**：7个问题，依次回答。每个回答后系统只复述关键词，不问为什么。

**问题**：

| 顺序 | 问题                                           |
| ---- | ---------------------------------------------- |
| 1    | “今天，你想跟它说点什么吗？”                   |
| 2    | “如果它有形状或颜色，它像什么？”               |
| 3    | “它想保护你什么？它在怕什么？”                 |
| 4    | “如果不被它带着走，你会怎么做？”               |
| 5    | “过去有没有一次，它没带走你？那次发生了什么？” |
| 6    | “如果对现在的自己说一句话，你会说什么？”       |
| 7    | “想把这句话存到‘一句声音’里吗？”               |

**交互流程**：
1. 系统显示第1个问题
2. 用户输入
3. 系统复述关键词：“你说了：[关键词]”
4. 系统显示下一个问题
5. 重复至结束

**结束语**：“你走完了这条路。记录在‘我的记录’里。”

**存储**：完整对话保存到“我的记录”。

---

#### 3.2 和它说说话

**形式**：与“心里的声音”对话，系统扮演声音回应（预设库）。

**流程**：
1. 系统：“你想对它说什么？”
2. 用户输入
3. 系统从预设库选一条回应
4. 系统：“还想说什么吗？”（最多3轮）
5. 结束：“你们说完了。可以保存。”

**预设回应库**（10条，随机）：
- “我听到了。我这样做，是怕你受伤。”
- “我不知道别的方式。”
- “你可以不听我的。”
- “我在这里很久了。”
- “谢谢你听我说。”
- “有时候我也累。”
- “我怕你受伤，也怕你不需要我。”
- “我不是你的敌人。”
- “可以让我小声一点。”
- “也许可以一起试试别的。”

---

#### 3.3 留一封信

**写给过去的自己**：
- “如果想给过去的自己写封信，你会写什么？”
- 文本框。保存到“我的记录”。

**写给未来的自己**：
- “想对未来的自己说什么？”
- 选择时间（1年/3年/5年）
- 文本框。保存。到时提醒：“有一封旧信。”

### 模块四：灯室（成长记录）

#### 4.1 一句声音

**功能**：录一句想对自己说的话。

**预设短句**（可选）：
- “慢慢来。”
- “可以怕，也可以不怕。”
- “已经做了不少。”
- “没事的。”

**背景音**（自选）：雨声 / 流水 / 风声 / 炉火 / 夜虫 / 安静

**使用**：在“来了什么”页面可播放。

---

#### 4.2 我的记录

**包含**：
- “走一段路”的对话
- “和它说说话”的对话
- “留一封信”的信件
- “小记”的记录
- 用户续写的故事

**交互**：按时间排列。可编辑，可删除。可导出。

**故事开头**（可选）：
> “每个人心里都有故事。你的故事，可能从‘怕’开始——”

用户可续写。

---

#### 4.3 时间图

**基于“小记”记录**：
- 横轴：月份
- 纵轴：小记次数
- 标注：哪些节气小记较多

**图上文字**：“小记多出现在[季节]。天地在陪你。”

**显示条件**：至少有3条小记。

---

### 模块五：我（数据）

#### 5.1 本地说明
> “所有记录只留在这里。”

#### 5.2 数据统计
- 使用天数
- 记录次数
- 小记次数
- 对话次数

#### 5.3 导出数据
- 按钮：【导出】
- 格式：JSON文件
- 提示：“数据在你手里。”

#### 5.4 一键清空
- 按钮：【清空】
- 确认：“确定吗？不可恢复。”
- 清空后重置

#### 5.5 版本切换
- 命理版 / 非命理版
- 提示：“语言会变，数据不变。”

#### 5.6 关于
- 版本：6.0
- 介绍：“心镜灯 · 顺着天地的自我对话”
- 边界：“这不是心理咨询。如果你感到持续痛苦，可以寻求专业帮助。”

## 五、用户旅程示例

1. 打开首页：“今日谷雨。有些感觉来了，也可以走。”

2. 天性小像：“你像一棵树。”

3. 身体感受：点“肩膀”。

4. 来了什么：点“恐惧”。系统：“它来了。它来的时候，身体哪里感觉到了？”用户选了“肩膀”。

5. 它想说什么：输入“你别对我太好”。系统：“那不是你，是心里一个声音。它可能想：怕靠太近、怕欠人情、怕自己不够。”

6. 一周后看一周图：发现周三有个✨。点开：那天用户记录了一次小记。

7. 走一段路：夜里走了7个问题。第6步对自己说：“已经做了不少。”存到了“一句声音”。

8. 我的记录：用户续写故事：“……原来慢一点也可以。”
