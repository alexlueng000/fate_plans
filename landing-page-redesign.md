# 落地页重新设计实施计划

## Context（背景）

易凡文化产品已从单一的八字对话功能扩展为三大核心功能：
1. **八字** - 精准计算命盘，AI深度解读人生轨迹
2. **心镜灯** - 每日情绪追踪，五行性格分析，发现情绪模式
3. **六爻** - 一事一卦，看清趋势、识别风险、获得行动建议

**当前问题**：
- 现有落地页（`fate-frontend/app/page.tsx`）只突出八字功能
- 心镜灯和六爻完全没有展示，用户无法了解完整产品价值
- 文案偏传统，缺乏现代感和情感共鸣
- 转化路径单一，缺乏分层引导

**设计目标**：
- 平等展示三大功能，让用户理解产品的完整价值
- 用现代化、有温度的文案建立情感连接
- 设计多层次转化漏斗，降低用户决策门槛
- 保持东方克制美学（朱砂红 + 宣纸白 + 茶金）

## Design Principles（设计原则）

1. **东方克制美学**
   - 保持现有的色彩系统（朱砂红 + 宣纸白 + 茶金）
   - 使用极简圆角（2-4px）
   - 留白充足，避免视觉拥挤

2. **功能平等展示**
   - 三大功能使用相同的卡片尺寸和布局权重
   - 通过色彩区分功能，但不分主次
   - 每个功能都有独立的CTA按钮

3. **情感共鸣优先**
   - 文案从用户痛点出发
   - 使用真实场景和故事
   - 强调"认识自己"而非"算命"

4. **渐进式引导**
   - Hero Section：建立兴趣（查看示例）
   - 功能区：深入了解（体验功能）
   - 场景区：情感共鸣（我也需要）
   - 评价区：建立信任（真实评价）
   - 最终CTA：转化注册（免费开始）

5. **性能优先**
   - 图片懒加载
   - 动画使用 CSS transform（GPU加速）
   - 组件按需加载
   - 避免不必要的重渲染

   - 特别是三大功能的价值主张描述
   - 确保符合品牌调性（理性、科学、温暖）

3. **可视化组件**
   - 八字预览：使用简化版，不需要完整的排盘功能
   - 情绪图表：使用示例数据，不需要真实API调用
   - 卦象预览：使用静态示例，不需要真实起卦逻辑

4. **响应式优先**
   - 移动端用户占比高，优先保证移动端体验
   - 功能区在移动端改为垂直堆叠
   - 图表在移动端简化展示

5. **A/B测试准备**
   - CTA文案可能需要测试（"免费开始" vs "立即体验"）
   - 功能展示顺序可能需要测试（八字优先 vs 心镜灯优先）
   - 预留数据埋点位置
