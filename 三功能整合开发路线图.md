# 三功能整合开发路线图 v2.0

**项目目标**：在现有八字对话基础上，整合**心镜灯（心迹）**和**六爻玄机**，形成完整的命理+疗愈+占卜平台。

**文档版本**：2.0  
**创建日期**：2026-04-24  
**预计工期**：15-20天

---

## 总体架构

```
fate 平台
├─ 八字对话（已完成）
│  └─ /chat 或 /panel
├─ 心镜灯（新增）
│  └─ /xinji
└─ 六爻玄机（新增）
   └─ /liuyao
```

**导航方案**：
- **桌面端**：顶部横向 Tab（八字对话 | 心镜灯 | 六爻玄机）
- **移动端**：底部固定 Tab（💬八字 | 📖心迹 | 🎲六爻）
