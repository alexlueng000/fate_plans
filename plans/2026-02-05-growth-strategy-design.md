# 八字产品增长策略设计

> 创建日期：2026-02-05
> 状态：待实施

## 背景与目标

### 产品现状
- **阶段**：早期增长期（100-1000 DAU）
- **主要瓶颈**：
  - 留存率低 - 用户来了但很快流失
  - 产品差异化不足 - 市场上类似产品多

### 增长目标
在以下四个维度实现全面增长：
1. 用户规模增长
2. 收入增长
3. 产品功能增长
4. 技术能力增长

## 核心策略：内容为基，服务为核，社交为翼

采用**阶梯式增长引擎**，分三个阶段推进：

```
第一阶段（立即启动）     第二阶段（1-2个月后）    第三阶段（3个月后）
      ↓                        ↓                      ↓
   内容驱动                 深度服务               社交裂变
   建立习惯                 提升付费               规模获客
```

---

## 第一阶段：内容驱动 - 建立用户习惯

### 目标
解决留存问题，让用户养成每天打开产品的习惯

### 核心功能清单

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| **每日运势** | 基于用户八字生成当日运势提示 | P0 |
| **节气解读** | 24节气特别内容，结合用户八字 | P0 |
| **流月分析** | 每月初推送当月运势概览 | P1 |
| **命理小知识** | 每日一条八字/风水知识科普 | P1 |
| **历史名人八字** | 名人八字故事，增加趣味性 | P2 |

### 推送触达策略

```
早晨 8:00  →  每日运势推送（微信服务号/小程序订阅消息）
节气当天  →  节气特别解读
每月 1 日 →  流月运势概览
```

### 技术实现要点

- 后端：新增 `/api/daily-fortune` 接口，基于用户八字 + 当日干支计算
- 前端：首页增加"今日运势"卡片模块
- 小程序：接入订阅消息能力，用户授权后定时推送

### 成功指标
- D7 留存率 > 20%
- 每日运势查看率 > 40%

---

## 第二阶段：深度服务 - 建立差异化与付费转化

### 目标
通过独特的深度服务建立竞争壁垒，提升付费转化率

### 核心功能清单

| 功能 | 描述 | 付费模式 | 优先级 |
|-----|------|---------|-------|
| **流年详解** | 某一年的详细运势分析 | 单次付费 | P0 |
| **大运十年** | 某步大运的深度解读 | 单次付费 | P0 |
| **八字合婚** | 两人八字匹配度分析 | 单次付费 | P1 |
| **择日服务** | 婚嫁/开业/搬家吉日推荐 | 单次付费 | P1 |
| **宝宝起名** | 基于八字的姓名推荐 | 单次付费 | P2 |
| **VIP 会员** | 无限次深度分析 + 专属客服 | 订阅制 | P2 |

### 付费定价策略

```
免费层：基础八字解读 + 每日运势
├── 单次付费：流年详解 ¥9.9 / 大运十年 ¥19.9 / 合婚 ¥29.9
└── VIP 会员：¥99/年（无限次深度分析）
```

### 差异化要点

1. **AI 深度解读**：基于 RAG 知识库的个性化分析，非模板填充
2. **持续跟踪**：用户可保存多个命盘（自己、家人、朋友），形成"命理档案"
3. **专业背书**：结合子平/盲派理论，输出内容有理有据

### 成功指标
- 付费转化率 > 5%
- ARPU（用户平均收入）> ¥10

## 第三阶段：社交裂变 - 低成本规模获客

### 目标
通过社交功能实现用户自传播，降低获客成本

### 核心功能清单

| 功能 | 描述 | 裂变系数 | 优先级 |
|-----|------|---------|-------|
| **八字卡片分享** | 生成精美个人八字卡片 | 高 | P0 |
| **好友八字对比** | 邀请好友查看两人八字相性 | 高 | P0 |
| **邀请有礼** | 邀请新用户，双方获得奖励 | 中 | P1 |
| **命理话题圈** | 用户交流八字心得 | 低 | P2 |
| **排行榜/成就** | 八字稀有度排行、成就徽章 | 中 | P2 |

### 裂变机制设计

```
用户 A 分享八字卡片
       ↓
好友 B 扫码查看 → 注册并输入自己的八字
       ↓
系统生成 A 与 B 的八字对比报告
       ↓
A 获得积分奖励，B 获得免费基础解读
       ↓
B 继续分享，形成裂变循环
```

### 分享内容设计要点

1. **视觉吸引**：卡片设计精美，符合东方美学
2. **信息克制**：只展示部分信息，引导好奇心
3. **社交货币**：内容有趣/有料，分享能体现用户品味

### 成功指标
- 分享率 > 10%
- 裂变系数 > 1.2
