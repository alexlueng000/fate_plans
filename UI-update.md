# UI 极简高端改造计划

## Context

当前应用采用小红书风格的消费级视觉语言（亮红 #FF2442、大圆角、渐变、发光特效）。用户希望邀请世界顶级 UI 专家对界面进行改造，目标：**高端大气、少用圆角、极简风格**。

参照 10 位顶级专家的核心理念：
- **Kenya Hara（MUJI）**：留白是主动的设计语言
- **Dieter Rams**：少即是多，每个元素必须有存在理由
- **Massimo Vignelli**：网格纪律，结构先于装饰
- **Jony Ive**：精准的材质感与细节深度
- **Tadao Ando**：东方极简主义，沉默即表达

目标是将应用从"社交 App"气质转向"高端咨询服务/奢侈品刊物"气质。

## 设计决策

### 色彩体系（全面重构）

#### 四层色彩架构

```
一级：品牌层     朱砂红 / 暗朱砂          → 按钮、链接、强调元素
二级：空间层     宣纸白 / 暖米白 / 浅沙色  → 页面背景、区块底色
三级：内容层     墨灰 / 棕灰 / 淡灰褐      → 标题、正文、次级文字
四级：点缀层     茶金 / 雾青               → 图标、标签边框、分隔线微点缀
```

> **核心原则**：降饱和、收色数、统一暖调系统。不要每个模块自己决定颜色。

### 圆角体系（大幅削减）

| 变量 | 旧值 | 新值 |
|------|------|------|
| `--radius-sm` | `6px` | `2px` |
| `--radius-md` | `10px` | `3px` |
| `--radius-lg` | `14px` | `4px` |
| `--radius-xl` | `20px` | `4px` |（大卡片变方正） |
| `--radius-2xl` | `28px` | `4px` |
| `--radius-full` | `9999px` | 保留（仅用于头像/徽章） |

### 特效删减

- **删除**：所有 glow 效果（`shadow-glow-red`、`shadow-glow-gold`）
- **删除**：所有装饰性动画（float、pulse-glow、rotate-slow、shimmer）
- **保留**：功能性动画（fadeIn、scaleIn 用于下拉菜单）
- **删除**：渐变背景（Landing Hero 背景、按钮渐变改为纯色）
- **简化**：阴影系统，统一为暖灰阴影（非冷灰）：
  - 轻：`0 1px 4px rgba(60,40,20,0.06)`
  - 中：`0 2px 12px rgba(60,40,20,0.08)`
  - 禁止：蓝色 focus glow（改用低透明度朱砂红）

### 材质感（轻微，不过度）

> 高级感 = 颜色 + 留白 + 材质 + 控制欲。不是换皮，是换体系。

- **米白背景**：可叠加极轻微纸张噪点（opacity ≤ 0.03），增加宣纸感
- **深朱砂区块**：可加极浅东方暗纹（opacity ≤ 0.05），避免大面积纯色平淡
- **卡片阴影**：用暖灰而非标准黑灰阴影
- **金色**：不发光，只做平面点缀，不大面积铺

## 修改优先级

### P0（先改，不改没救）

1. **主红换成朱砂红 `#B54434`**：基调不改，一切都是换皮
2. **去掉高饱和蓝/黄/橙卡片**：廉价感主要来源
3. **所有白底换成暖米白体系 `#F7F3EE`**：整体立刻稳下来
4. **文字从纯黑改成墨灰 `#2A2522`**：高级感提升一个台阶

### P1（进一步提升质感）

5. **加入茶金 `#B08A57` 作为极少量强调色**：器物感、文化感
6. **底部大红块改成深朱砂 `#A13D30` + 极轻纹理**：避免硬广感
7. **表单输入框边框从冷灰改成暖灰 `#E2D8CE`**：现在还是互联网表单感
8. **Focus 态改用低透明度朱砂红**：禁止蓝色 glow

### P2（材质层次，真正的高级）

9. **米白背景加极轻噪点纹理**（opacity ≤ 0.03）
10. **深色区块加极浅东方暗纹**（opacity ≤ 0.05）
11. **阴影全面换成暖灰色系**

## 修改顺序

1. `globals.css` — 建立新设计基础，自动影响大部分组件
2. `Header.tsx` — 最高曝光率
3. Landing `page.tsx` — 第一印象
4. Chat 组件（MessageList → InputArea → QuickActions → SimplifyPanel）
5. `panel/page.tsx`
6. `Footer.tsx`
